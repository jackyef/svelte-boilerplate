Server-side rendering (SSR)

This boilerplate also include server-side rendering server written in koa! No need to configure anything else. You only need to run:

## start the webpack-dev-server that serve the static assets
yarn dev:client

## then run our koa SSR server
yarn dev:server

And, that's it!

If there's anything you would like to see in this boilerplate, feel free to create an issue at the github repo.

If you want to contribute, that will be great! Simply create a pull request to the repo!
